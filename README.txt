ระบบเช็คชื่อด้วยการสแกนใบหน้า (Face Recognition Attendance System)
=================================================================

1. การติดตั้ง (สำหรับ Windows)
------------
1.1 ติดตั้ง Python:
    - ดาวน์โหลด Python 3.9 หรือ 3.10 จาก python.org
    - เลือก "Add Python to PATH" ตอนติดตั้ง

1.2 ติดตั้ง Package ที่จำเป็น:
    เปิด Command Prompt (Admin) แล้วรันคำสั่งต่อไปนี้:
    
    pip uninstall opencv-python
    pip uninstall opencv-contrib-python
    pip cache purge
    pip install opencv-contrib-python==4.8.1.78

1.3 ทดสอบการติดตั้ง:
    python -c "import cv2; print(cv2.__version__)"

1.4 การรันโปรแกรม:
    - เปิด Anaconda Prompt
    - รัน: conda activate face_recognition
    - ไปที่โฟลเดอร์โปรเจค: cd path\to\PROJECT-TEST
    - รันโปรแกรมตามขั้นตอนในหัวข้อ 2

*******************************************************************************************************************************
2. การใช้งานโปรแกรม
------------------
2.1 การเก็บรูปภาพนักศึกษา (capture_faces.py)
    - รันโปรแกรม: python capture_faces.py
    - ใส่รหัสนักศึกษาเมื่อโปรแกรมถาม
    - รอให้กล้องตรวจจับใบหน้า (จะเห็นกรอบสี่เหลี่ยมสีฟ้า)
    - กด 'c' เพื่อถ่ายรูป (ต้องถ่าย 5 รูป)
    - กด 'q' เพื่อออกจากโปรแกรม

2.2 การสร้างฐานข้อมูลใบหน้า (encode_faces.py)
    - รันโปรแกรม: python encode_faces.py
    - รอจนกว่าโปรแกรมจะประมวลผลเสร็จ

2.3 การเช็คชื่อแบบ realtime (recognize_realtime.py)
    - รันโปรแกรม: python recognize_realtime.py
    - ระบบจะเริ่มตรวจจับและบันทึกการเข้าเรียนอัตโนมัติ
    - กด 'q' เพื่อออกจากโปรแกรม

3. คำแนะนำเพิ่มเติม
------------------
- ควรถ่ายรูปในที่ที่มีแสงสว่างเพียงพอ
- ขณะถ่ายรูปควรหันหน้าตรงกล้อง
- เมื่อถ่ายรูปแล้วจะถูกบันทึกในโฟลเดอร์ dataset
- การเช็คชื่อจะถูกบันทึกในฐานข้อมูล attendance.db

4. การแก้ไขปัญหาเบื้องต้น
----------------------
- หากกล้องไม่ทำงาน ให้ตรวจสอบการเชื่อมต่อกล้อง
- หากบันทึกรูปไม่ได้ ให้ตรวจสอบสิทธิ์การเข้าถึงโฟลเดอร์
- หากตรวจจับใบหน้าไม่ได้ ให้ปรับแสงสว่างให้เพียงพอ
